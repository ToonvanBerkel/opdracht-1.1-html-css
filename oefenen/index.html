HTML

uitleg

HTML, oftewel HyperText Markup Language, is de standaardtaal voor het maken en structureren van webpagina's. Het is een essentieel onderdeel van het web omdat het de basis vormt voor het weergeven van tekst, afbeeldingen, links en andere media in browsers. HTML bestaat uit een reeks tags (etiketten) die informatie verschaffen aan de browser over hoe de inhoud van een webpagina moet worden weergegeven.
